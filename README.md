> UML 软件分析设计与建模大作业 · 完整项目仓库

本仓库包含「启航AI」大学生职业生涯规划系统的 **全部开发产物**：源代码、三次迭代的设计文档与UML模型、成果汇报材料，以及课程案例教程。

## 项目简介

「启航AI」是一套面向在校大学生的智能职业生涯规划系统。系统依托大语言模型（LLM），通过多维度心理测评生成精准的用户画像，并结合行业数据为用户提供个性化的职业规划方案，覆盖从自我认知到求职落地的完整链路。

### 核心功能

| 模块 | 说明 |
|------|------|
| 登录认证 | 用户名密码登录，Token 认证机制 |
| 用户信息 | 学校/专业/GPA + 实习/项目/竞赛经历 CRUD |
| 用户画像 | MBTI 性格测试 / 霍兰德职业兴趣 / 胜任力评估 + 雷达图 |
| 招聘信息 | 20 条大厂岗位数据，支持筛选与收藏 |
| 职业规划 | 职业规划时间轴、就业方向推荐、岗位匹配、技能提升建议 |
| 模拟面试 | AI 驱动的多轮对话面试 + 评估报告 |
| 简历优化 | 上传简历 → AI 优化改写 → 三栏对比 → 评分 |
| 系统管理 | 用户管理、数据统计看板、系统日志 |

### 技术栈

- **前端**：Vue 3 + Vite + TypeScript + Pinia + Element Plus + ECharts
- **后端**：Django 4.x + Django REST Framework + SQLite
- **大模型**：Mock 模式（内置丰富模拟回复）/ 支持接入任意 OpenAI 兼容 API

## 快速启动（源代码）

详细的环境配置和启动步骤请参见 [qihang-ai-demo/README.md](qihang-ai-demo/README.md)。

**简要步骤：**

```bash
# 1. 进入代码目录
cd qihang-ai-demo

# 2. 启动后端
cd backend
pip install -r requirements.txt
python seed.py                    # 初始化演示数据
python manage.py runserver        # http://localhost:8000

# 3. 启动前端（新终端）
cd frontend
npm install
npm run dev                       # http://localhost:3000
```

**演示账号：**

| 角色 | 用户名 | 密码 |
|------|--------|------|
| 学生 | `demo_student` | `demo1234` |
| 管理员 | `admin` | `admin1234` |
